

Diese To-Do App ist eine Kommandozeilen-Anwendung, die ich mit Python und SQLite entwickelt habe.
Benutzer können sich registrieren, einloggen und ihre eigenen Aufgaben verwalten

## Funktionen

- Benutzer registrieren
- Login mit gehashtem Passwort (SHA256)
- Neue Aufgaben ersellen
- Aufgaben anzeigen
- Aufgaben als erledigt markieren
- Aufgaben löschen

## Technologien
ein Login-Systeme
- Python 3
- sqlite3 (für die Datenbank)
- hashlib (für Passwort-Hashing mit SHA256)

## Was ich mit dem Projekt gelernt habe

Mit diesem Projekt wollte ich verstehen, wie:
- man mit einer Datenbank arbeitet (SQLite)
- ein Login-System, funktioniert
- Passwörter sicher gespeichert werden
- CRUD-Funktionen (Create, Read, Update, Delete) umgesetzt werden
- man ein strukturiertes CLI-Programm aufgebaut

## Starten des Projekts

1. Repository klonen
2. In den Projektordner wechseln
3. Programm starten mit

python main.py

Die Datenbank wird automatisch erstellt.
